# // fonts

## Display & Brand — Space Mono
- Weights: 400 (Regular), 700 (Bold)
- Source: https://fonts.google.com/specimen/Space+Mono
- License: SIL Open Font License 1.1
- Usage: Logo mark, headlines, code specimens, labels, taglines

## Body & Supporting — DM Sans
- Weights: 300 (Light), 400 (Regular), 500 (Medium)
- Source: https://fonts.google.com/specimen/DM+Sans
- License: SIL Open Font License 1.1
- Usage: Body copy, descriptions, proposals, UI body text

## Google Fonts import

```css
@import url('https://fonts.googleapis.com/css2?family=Space+Mono:wght@400;700&family=DM+Sans:wght@300;400;500&display=swap');
```
