## Files

| File | Description | Use when |
|------|-------------|----------|
| `mark-dark.svg` | // mark, Signal on Void, rounded rect | Dark UI, social avatar, app icon |
| `mark-light.svg` | // mark, Void on Parchment, rounded rect | Print, light UI, inverted contexts |
| `mark-signal-transparent.svg` | // mark, Signal, transparent bg | Overlaying on dark photos or surfaces |
| `mark-dark-transparent.svg` | // mark, Void ink, transparent bg | Overlaying on light surfaces |
| `lockup-horizontal-dark.svg` | // + forwardslash + DEVELOPMENT, dark bg | Website header, email header, dark contexts |
| `lockup-horizontal-light.svg` | // + forwardslash + DEVELOPMENT, light bg | Print, proposals, light contexts |
| `favicon.svg` | 32×32 mark, Signal on Void | Browser favicon, PWA icon |

## Usage Rules

- Always use `//` — never a single `/`
- Never stretch, skew, or rotate the mark
- Never add drop shadows, outlines, or glow effects
- Minimum size: 16px digital / 6mm print
- At small sizes, use the mark alone — not the full lockup
- Maintain clearspace equal to the cap-height of `//` on all sides
